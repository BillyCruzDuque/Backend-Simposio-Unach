# Backend Node Simposio 💻
## Descripción 📚
Backend utilizado para el proyecto de registro del Simposio Unach 2022.
## Instalación💡
    npm install 
## Ejecución Desarrollo 🚀
    npm start:dev

## Ejecución Producción 🚀
    npm start

## Caracteristicas 🪐
- Server
- RestApi
- Modular

## Tecnologias 🛠
- NodeJS
- Express
- MongoDB
- Mongoose
- JWT
- Dotenv
- Cors
- Nodemon
- Git
- Github
- Postman
